## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Redemption19/c-stone.git
```

2. Install dependencies
```bash
cd c-stone
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
